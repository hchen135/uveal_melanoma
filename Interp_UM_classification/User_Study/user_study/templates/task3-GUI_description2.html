Task 3: UM prognostication with cytopathology images, AI recommendation and AI extracted infromation

In this page, we will introduce the second part of GUI in task 3.

With the pie chart, the AI makes recommendations according to the cell distribution in the pie chart following a rule set.

  • Obeying the rule set makes the AI system predicting GEP class 2 like.
  • Violating the rule set makes the AI system predicting GEP class 1 like.
  • The entire rule set criterion for AI to make diagnosis can be found in the pop out window by clicking the "Entire AI decision role" button.
